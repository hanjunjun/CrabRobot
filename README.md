# CrabRobot助手
## 包含功能
- 自动小宠物升到满级60级
- 角色掉线或游戏崩溃自动启动游戏输入账号密码和暴雪令牌重连
- 后台自动录像
- 掉线或被踢出刷经验区域会发送邮件提醒
